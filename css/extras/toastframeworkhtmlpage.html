Toast

A simple CSS framework

Toast is a super simple CSS framework, inspired by the likes of Twitter's Bootstrap and the 960 grid system to make rapid web design and development easier.

It includes the following components, which users are free to mix, match and modify to their needs:

  • An easy to use 12-column grid system
  • A baseline-friendly typographic stylesheet
  • Object Oriented buttons and messages
  • Beautiful and user friendly form presets

Object Oriented CSS

Inspired by Luke Beard's Object Oriented approach when designing Zerply, Toast includes a selection of OOCSS components - namely, buttons and messages.

Notice something cool about those components? When you hover over them, there's a lovely custom tooltip. To add a tooltip to an element, simply add a data-toastip attribute to it. For example, this little phrase has data-toastip="Hello, world!" applied to it.

Forms

Forms in Toast are super simple. Just write your labels and inputs as you normally would, and they'll get the styles below. If you don't want the default Form styles, simply remove the Forms CSS file from the Toast framework.

Grids

The grid system is just as simple to use as everything else. Here's the markup for the structure of this page:

<div class="wrap">
    <div class="grids">
        <div class="grid-8">
            There's a bunch of content in here!
        </div>
        <div class="grid-4">
            And a sidebar here!
        </div>
    </div>
</div>

The grid is made up of 12 columns, totalling to 960px full width. It's also responsive, and it'll snap down to one full width column below 700px.

Less CSS

Toast was built using LESS - it's a CSS preprocessor made to help speed up development with the addition of variables and other cool stuff. The LESS files are all included in the framework, but have been compiled into .css files ready for instant use. You should avoid compiling LESS files on-the-fly by processing them locally before using them on a website.
